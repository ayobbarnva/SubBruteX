## 📖 About

SubBruteX is a simple and lightweight subdomain brute-forcing tool written in Python.

The project is designed to perform subdomain enumeration using a wordlist while providing a clean and user-friendly command-line interface.

Its primary goal is to offer a straightforward and easy-to-understand implementation without unnecessary complexity.
